# sugalaa (demo UI)

This repo contains a **static, screenshot-friendly** demo UI that resembles the “Машины Сугалаа” reference you sent.

## Pages

- `index.html` — interactive demo
- `admin.html` — dashboard-like screen
- `admin.html#transactions` — import/view transactions (admin-only)

## Run locally

From the project folder:

```bash
cd /Users/jooy/Development/sugalaa
python3 -m http.server 5173
```

Then open:

- `http://localhost:5173/index.html`
